# LIFX & Flic Hub For Raspberry Pi

### Packages
- The standard C/C++ libraries, which should be installed by default on most Linux distributions.
- Python 3+

### Running

Execute `./setup.sh` and follow instructions.  
Execute `./start.sh`.
